Factorial
=========

Overview
--------

Factorial aims to provide an HTML/CSS editor with power user features with an emphasis on creating and maintaining valid layouts.

MAP Features
------------

* Layouts
  * Spring
  * Horizontal
  * Vertical
  * Grid
* Discovery
* CSS editing of an element
* Selectorgadget
* Extract component
* Constraints
  * Grid
* Save
  * Diff
  * Full project
  * Annotations
  
UI Components
-------------

* Ribbon menu
* inner
* overlay
* springs
